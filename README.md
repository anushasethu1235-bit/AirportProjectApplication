# AirportProjectApplication
using core-java , spring-boot , hybernate , mysql and postmal tool
Description : The Airport Baggage Storage and Reporting System is a backend application designed to store and manage passenger baggage information and automatically share baggage details with connected flights at their destination airports. The system supports secure and real-time baggage tracking across terminals and ensures smooth communication between airport systems and airline databases.
 
Roles and Responsibilities:
•	Developed RESTfulAPIs using SpringBoot and SpringMVC to handle baggage check-in, updates, and reporting to destination flights.
•	Used Java for implementing business logic such as baggage assignment, routing, and validation rules.
•	Integrated with MySQL using Hibernate and Spring Data JPA, and implemented pagination for efficient handling of large baggage records.
•	Applied Spring Security to secure API endpoints and enforce role-based access control or internal airport staff.
•	Created modules to generate and transmit structured baggage reports to corresponding flights at various destination airports.
•	Managed entity relationships and optimized database queries to ensured at a consistency and retrieval performance.
•	Followed Agile methodologies, participated in sprint planning, daily stand-ups, and collaborated closely with QA and frontend teams for seamless development.
